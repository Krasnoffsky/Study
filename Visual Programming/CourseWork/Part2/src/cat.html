Вернуться в меню

Категории

При нажатии на кнопку 'Категории' будет показано меню с 12 полями. Каждое поле ограничивает выводимые рецепты. В меню есть возможность выбрать одну из 11 категорий блюд или уставновить вывод всех доступных рецептов. При выборе категории в поле с именами рецептов будут отображаться только те рецепты, которые относятся к выбранной категории. Если установлен вывод только избранных рецептов, то будут выводиться рецепты, относящиеся к данной категории и являющиеся избранными. По умолчанию стоит отображение всех рецептов.

Редактирование рецептов

При нажатии на кнопку 'Изменить рецепт', будет открыто окно редактирования выбранного рецепта. Вся информации о рецепте уже введена. В поле (1) указано название рецепта. В поле (2) указаны ингредиенты рецепта. В поле (3) указан рецепт приготовления. Также отображается сопутсвующее изображение. Данные в полях доступны для редактирования. При нажатии на кнопку (4) будет предложено выбрать новое изображение для вашего рецепта. Чтобы сохранить изменения нажмите на кнопку (5). Чтобы отменить редактирование нажмите на кнопку (6) или закройте окно редактирования.

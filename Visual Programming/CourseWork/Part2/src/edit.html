Редактирование рецептов

При нажатии на кнопку 'Изменить рецепт', будет открыто окно редактирования выбранного рецепта. Вся информации о рецепте уже введена. В поле под изображением указано название рецепта. В поле справа от изображения указаны ингредиенты рецепта. В поле под ингредиентами указан рецепт приготовления. Также отображается сопутсвующее изображение. Данные в полях доступны для редактирования. При нажатии на кнопку 'Загрузить изображение' будет предложено выбрать новое изображение для рецепта. Чтобы сохранить изменения нажмите на кнопку 'Сохранить рецепт'. Чтобы отменить редактирование нажмите на кнопку 'Отменить' или закройте окно редактирования.

Создание рецептов

При нажатии на кнопку 'Новый рецепт', появится окно создания нового рецепта. В данном окне необходимо заполнить все поля, иначе добавлени рецепта будет невозможно. При попытке сохранить рецепт, не заполнив все поля, прмложение уведомит о том, что не все поля заполнены. В поле 'Название' нужно написать название вашего рецепта. В поле 'Ингредиенты' нужно написать ингредиенты, необходимы для приготовления вашего рецепта. В поле 'Рецепт' необходимо написать рецепт. При нажатии на кнопку 'Категория' появится меню с 11 категориями блюд. Необходимо выбрать одну из категорий, к которой относится блюдо, приготовленное по вашему рецепту. При нажатии на кнопку 'Загрузить изображение' откроется окно выбора изображение. Выберете сопроводительное изображение для вашего рецепта. Изображение является единственным не обязательным полем. Если не будет выбрано изображение, установится изображение по умолчанию. Нажмите на кнопку 'Сохранить рецепт' для сохранения рецепта. Чтобы отказаться от создания нового рецепта, нажмите на кнопку 'Отменить' или закройте окно. Если закрыть окно создания нового рецепта, то все несохранённые данные будут утеряны и создания нового рецепта придётся начать сначала.

Создание рецептов

При нажатии на кнопку 'Новый рецепт', появится окно создания нового рецепта. В данном окне необходимо заполнить все поля, иначе добавлени рецепта будет невозможно. при попытке сохранить рецепт, не заполнив все поля, программа уведомит о том, что не все поля заполнены. В поле (1) нужно написать название вашего рецепта. В поле (2) нужно написать ингредиенты, необходимы для приготовления вашего рецепта. В поле (3) необходимо написать рецепт. При нажатии на кнопку (4) появится меню с 11 категориями блюд. Необходимо выбрать одну из категорий, к которой относится блюдо, приготовленное по вашему рецепту. При нажатии на кнопку (5) откроется окно выбора изображение. Выберете сопроводительное изображение для вашего рецепта. Изображение является единственным не обязательным полем. Если не будет выбрано изображение, установится изображение по умолчанию. Нажмите на кнопку (6) для сохранения рецепта. Чтобы отказаться от создания нового рецепта, нажмите на кнопку (7) или закройте окно. Если закрыть окно создания нового рецепта, то все несохранённые данные будут утеряны и создания нового рецепта придётся начать сначала.

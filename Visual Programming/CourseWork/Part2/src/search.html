Поиск

При вводе текста в поле (1) будет начат поиск. Для поиска достаточно ввести текст, поиск происходит автоматически при редактировании текста в поле (1). Доступен поиск по фильтрам Для выбора фильтра нужно нажать на кнопку 'Фильтры' (2). Будет открыто меню (3) с 4 доступными фильтрами. Фильтр определяет где имеено буде происходить поиск по тексту их поля (1). При выборе фильра 'Название', поиск текста будет происхожить в названиях рецепта. Аналогично работают остальные фильтры. При выборе фильтра 'Глобальный поиск', будет осуществляться поиск заданного текста в названиях, ингредиентах и с тексте рецепта. По умолчанию стоит фильтр 'Глобальный поиск'. Все найденные результаты будут отображены в поле с названиями рецептов (3). Если поле пустое, рецептов, соответсвующим критериям поиска, найдене было. Если в результате поиска были найдены рецепты, можно выбрать к отображение по категории и отображение избранных рецептов.
